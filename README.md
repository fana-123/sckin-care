# SHAAX Skincare Website

A bilingual (English/Somali) skincare e-commerce website featuring face wash products, moisturizers, serums, and masks.

## Live Website
Visit: https://fana-123.github.io/sckin-care/

## Features
- Responsive design
- Bilingual content (English/Somali)
- Product categories
- Interactive shopping cart
- Mobile-friendly navigation

## Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - pure HTML/CSS/JS

## Pages
- `index.html` - Homepage
- `face-wash.html` - Face wash products
- `moisturizers.html` - Moisturizer products
- `serums.html` - Serum products
- `masks.html` - Face mask products
- `cleansers.html` - Cleanser products
